Installer WildFly sur la machine

WildFly est un serveur d'applications Java EE qui permet de déployer des applications Spring Boot sous forme de fichiers WAR. Suivons les étapes pour installer WildFly en local.

Étapes d'installation

1. Télécharger WildFly

  • Télécharger WildFly

2. Installer WildFly

Windows

  1. Téléchargez et décompressez le fichier zip.

  2. Placez le dossier décompressé dans un emplacement clé comme C:\wildfly.

  3. Dans le dossier bin, lancez

    standalone.bat

    pour démarrer WildFly.

Mac/Linux

  1. Ouvrez un terminal et décompressez le fichier téléchargé avec la commande suivante :

    tar -xvf wildfly-33.0.2.Final.tar.gz

  2. Déplacez le dossier décompressé vers un emplacement comme /opt/wildfly.

  3. Pour démarrer WildFly, exécutez la commande suivante dans le dossier bin:

    ./standalone.sh

3. Ajouter WildFly aux variables d'environnement (optionnel)

Sur certaines plateformes, il est utile d'ajouter WildFly au PATH pour pouvoir l'exécuter depuis n'importe quel répertoire.

Windows

  1. Ouvrez les Paramètres système avancés.

  2. Cliquez sur Variables d'environnement.

  3. Ajoutez une nouvelle variable d'environnement avec le chemin de WildFly, par exemple C:\wildfly\bin.

Mac/Linux

  1. Ajoutez le chemin de WildFly dans votre fichier ~/.bashrc ou ~/.zshrc:

    export PATH=$PATH:/opt/wildfly/bin

4. Vérifier l'installation

Pour vérifier que WildFly fonctionne correctement, accédez à l'URL suivante dans votre navigateur :

http://localhost:8080

Cela devrait vous afficher la page d'accueil de WildFly, confirmant que le serveur est bien démarré.
